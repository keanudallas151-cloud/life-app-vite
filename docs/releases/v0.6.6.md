# v0.6.6 — Lint fixes, iOS & mobile polish

## Lint fixes
- Fixed useEffect missing dependency in InventorsInvestors.jsx (roleChoice stale closure — changed to `if (selectedRole) setRoleChoice(selectedRole)`).
- Fixed `crypto is not defined` error in useInventorsInvestorsData.js — changed to `window.crypto.randomUUID()`.
- Suppressed three `@next/next/no-img-element` warnings with targeted inline eslint-disable comments. Justified because next.config.js already sets `images: { unoptimized: true }`, so no optimisation benefit would be gained from swapping to `<Image>`.

## UI polish
- Loading screen Life. dot now uses a true circular `<span>` with `width: 0.4em / height: 0.4em` instead of the raw period character, matching the homepage dot behaviour.
- Reading mode tooltip improved: cleaner dark caret pointing directly at the button, correct mobile padding, minimum touch target on the dismiss button.

## Mobile & iOS
- Researched top 20 most-used phones worldwide (iPhone 16 Pro Max, iPhone 15/16, iPhone SE, Samsung Galaxy S24/A55/A35/A15, Xiaomi Redmi Note 13, Xiaomi 14, OPPO A58, Vivo Y100, Realme C65, OnePlus 12, Pixel 8, and others).
- Added dedicated CSS breakpoints for 360px (largest global Android segment), 375px (iPhone SE), 390–393px (iPhone 12/13/14/15 standard), 412px (Galaxy S Ultra / OnePlus / Pixel), and 430px (iPhone Pro Max).
- Added iOS-specific CSS: `-webkit-overflow-scrolling: touch` momentum scroll on all scroll containers, `-webkit-touch-callout: none` on images and cards, `-webkit-tap-highlight-color: transparent` on all interactive elements.
- Added `@media (display-mode: standalone)` rules for installed PWA and home screen use.
- Added Capacitor-compatible viewport and PWA meta for the native iOS packaging path via Capacitor.js (no language rewrite needed — the app is already JavaScript/React which runs in WKWebView).
- Strengthened `env(safe-area-inset-*)` guards at bottom nav, scroll containers, and fixed overlays.
- Improved touch target minimum sizes across cards and buttons to meet Apple HIG 44×44pt minimum.
